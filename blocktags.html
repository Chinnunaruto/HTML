Some Common Block Tags

hr

The <hr> element used to stand for "horizontal line" and that is still how it is typically displayed by browsers.However, in HTML5 the tag took on a semantic meaning,"semantic break".I will be using it often to break up example files when I wnt to display multiple tage in one file.

address

The <address> element indicates that the text in the address specific to the page,perhaps contact information for the author.

blockquote

The <blockquote> element is used for quotations.It should include the cite attribute in order to show or link to the source of the quote.

details

The <details> element is used in conjunction with the <summary> tag.It is one of the tags that was created in response to the repeated use of scripting to open and close content.It is not not supported by all of the browsers yet though.
